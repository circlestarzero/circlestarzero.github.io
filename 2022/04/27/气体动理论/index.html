第六章气体动理论

热力学系统与平衡态

热力学系统

  • 系统
    • 在给定范围内由大量宏观粒子所组成的宏观客体
  • 外界
    • 能够与所研究的热力学系统发生相互作用的其他物体
  • 分类
    • 孤立系统
      • 与外界无任何交换
    • 封闭系统
      • 无质量交换
    • 开放系统
    • 绝热系统
      • 与外界仅无热量交换

热力学平衡态

  • 定义
    • 一个系统在不受外界条件影响下,它的宏观性质不再随时间变化
    • 一定质量的气体,与外界无能量交换,内部无化学反应,核反应,仅由分子热运动使气体各部分达到: \(\rho,T,P\) 均匀的状态
  • tips
    • 一个孤立的系统的状态-必定会处于热力学平衡态
    • 实际上只是一种热动平衡

状态参量,状态图

状态参量(对平衡态的描述)

  • 分类
    • 几何参量 (\(V\) )
    • 力学参量 (\(P\) )
    • 化学参量 (\(v,M\) )
    • 电磁学参量 (\(E,B\) )
    • 热学参量 (\(T\) )
  • 分类
    • 微观量
      • 表征单个分子的性质与状态 (\(m,r,v\) )
    • 宏观量
      • 反映整个系统的宏观性质和状态 (\(P,T,V\) )
  • 分类
    • 广延量
      • 总系统 = 各个子系统之和 \((V,v)\)
    • 强度量
      • 总系统 = 各个子系统 \((\displaystyle T,P)\)

状态图

状态参量空间: 以独立的状态参量为坐标构成的一个空间

平衡态过程:系统在变化过程中经历每一状态都是平衡态

  • 平衡态过程可在状态空间上表示
  • 非平衡态,非平衡过程则不能用状态图表示

热力学过程

热力学过程: 系统从一个状态 经历热力学过程 到达另一个状态的过程

平衡过程:系统在变化过程中经历每一状态都是平衡态

  • 实际热力学过程任一状态都不是平衡态

尊静态过程: 一个过程其任一时刻的中间态都无限接近于一个平衡态

  • 非平衡态过程无线缓慢趋近于平衡态过程

对==准静态过程中间状态的平衡态==以及==系统的准静态变化过程==均可用状态图表示

分子相空间: 六维空间中的一点 (\(x,y,z,v_x,v_y,v_z\) )同一个分子的运动状态相对应,该点称为分子的代表点,这一六维空间称为分子相空间

  • 相空间的横坐标:位置
  • 纵坐标:速度(或动量)
  • 分子体系的某一瞬间状态 $$ 此时刻体系所有分子在相空间的一个确定分布
  • 相空间空间坐标 \(x,y,z\)
    • 空间坐标微元 \(\mathrm{d}x\mathrm{d}y\mathrm{d}z\)
    • 坐标空间体积元 : \(\mathrm{d}V=\mathrm{d}x\mathrm{d}v\mathrm{d}z\)
  • 相空间速度坐标 \(v_x,v_y,v_z\)
    • 速度坐标微元 \(\mathrm{d}v_x \mathrm{d}v_y \mathrm{d}v_z\)
    • 速度空间体积元 \(\mathrm{d}v=\mathrm{d}v_x\mathrm{d}v_y\mathrm{d}v_z\)
  • 分子相空间体积元: \(\mathrm{d}x\mathrm{d}y\mathrm{d}z\mathrm{d}v_x\mathrm{d}v_y\mathrm{d}v_z=\mathrm{d}^{3}\vec{v} \mathrm{d}^{3}\vec{r}\)

温度

当复合系统(有热接触)达到平衡时,两系统处于热平衡,具有相同的宏观性质:温度

温度决定一系统是否与其他系统处于热平衡

温标:温度的数字表示

  • 摄氏温标 \(t= 0\) ℃
  • 热力学温标 \(T=t+273.15 \mathrm{K}\)

热力学第零定律

  • 如果两个系统分别与第三个系统达到热平衡,则这两个系统彼此也处于热平衡状态

对理想气体的描述

理想气体的微观模型

  • 分子本身大小忽略不计
  • 分子间相互作用忽略不计
  • 分子间,分子与气壁的碰撞是弹性碰撞

理想气体: 大量不停的,无规则运动的,无引力的弹性质点的集合

组成理想气体的质点运动遵循经典力学规律

在压强较低,温度较高的情况下比较拟合

理想气体的状态方程

标况下气体摩尔体积 \(v_0=22.41 \times 10^{-3} \mathrm{m^{3} / mol}\)

气体普适常数: \(R = 8.31 \mathrm{J/mol \cdot K}\)

摩尔数: \(\displaystyle v = \frac{m}{M}\)

\(\displaystyle PV = vRT\)

\(\displaystyle PV = \frac{m}{M}RT\)

\(N_A= 6.02 \times 10^{23} / \mathrm{mol}\)

\(N\) 气体分子数

\(\displaystyle v = \frac{N}{N_A}\)

\(\displaystyle PV = \frac{N}{N_A} R T\)

玻尔兹曼常数 \(\displaystyle k=\frac{R}{N_A}=1.38\times 10^{-23} \mathrm{J/K}\)

\(PV = N kT\)

分子密度 \(\displaystyle n = \frac{N}{V}\)

\(\displaystyle P = nKT\)

理想气体的压强

统计规律: 单个事件看不出来规律,大量实验将出现的规律

概率: 某个事件出现的可能性的量度 \(P_i = \lim_{N \to \infty} \frac{N_i}{N}\)

概率的归一化条件 : \(P = \sum_{i} p_i = 1\)

各方向分子速率的统计平均值相等

  • \(\overline{v_x} = \overline{v_y} = \overline{v_z}\)

\[ \overline{v_x}=\frac{\sum_{i=1}^{N}v_{ix} }{N} \]

\(\displaystyle \overline{v_x^2} = \overline{v_y^{2} } = \overline{v_z^{2} }\)

\(\mathrm{d}t\) 时间内, 具有 \(v_i\) 的分子施于 \(\mathrm{d}A\) 的冲量

  • 体积 \(v_i \mathrm{d}t \mathrm{d}A\)
  • 分子数 \(n_iv_{ix}\mathrm{d}t \mathrm{d}A\)
  • 冲量 \(\mathrm{d}I_i=2m n_i v_{ix}^{2}\mathrm{d}t \mathrm{d}A\)

\[ \begin{aligned} \mathrm{d}I & = \sum_{i}\mathrm{d}I_i \\ & = \sum_{v_{ix}>0}2mmv_{ix}^{2}\mathrm{d}t \mathrm{d}A \\ & = \frac{1}{2}\sum_{i} m n_i v_{ix}^{2} \mathrm{d}t \mathrm{d}A\\ & = \sum_{i} m n_i v_{ix}^{2}\mathrm{d}t \mathrm{d}A \end{aligned} \]

\[ \begin{aligned} P & = \frac{\mathrm{d}I}{\mathrm{d}t \mathrm{d}A} \\ & = \sum_{i} m n_i v_{ix}^{2} \\ & = mn \frac{\sum_{i}n_i v_{ix}^{2} }{n} \\ & = m n \overline{v_x} \\ \end{aligned} \]

\(\displaystyle P = m n \overline{v_x}\)

分子的平均平动动能: \(\overline{\varepsilon _{kt} }=\frac{1}{2}m \overline{v^{2} }\)

\[ P = m n \overline{v_x^{2} } = m n \frac{1}{3} \overline{v^{2} } = \frac{2}{3} n \overline{\varepsilon_{k,t} }\]

\(P\) 的微观本质:

  • 压强的大小反映了分子平均平动动能的大小(分子数密度不变情况下)

理想气体的温度

理想气体分子的平均平动动能只与温度有关

温度是分子无规则运动剧烈程度的标志

温度是大量微观分子热运动的集体表现

\(\displaystyle \overline{\varepsilon _{k,t} }\neq 0,T \neq 0\)

方均根速率

\[ \overline{\varepsilon _{k} }= \frac{1}{2}m \overline{v^{2} } = \frac{3}{2} \frac{P}{n} = \frac{3}{2}kT\]

\[ \overline{v^{2} }= \frac{3kT}{m} \]

\[ \sqrt{\overline{v^{2} } } = \sqrt{\frac{3kT}{m} } = \sqrt{\frac{3RT}{M} }\]

分子速率的一种统计平均值

常温下气体分子速率与声波在空气中的传播速率等量级

能量均分定理, 理想气体的内能

自由度:决定一物质在空间的位置所需要的独立坐标数 通常用 \(i\) 表示

确定一质点的坐标:

平动自由度 \(i=t=3\)

确定一直线的位置

直线转动自由度 \(r=2\)

\(i=t+r=5\)

刚体转动自由度:\(r=3\) (绕轴)

双原子分子-非刚性(高温时) 振动自由度 \(S=1\)

双原子分子-非刚性(高温时) 振动自由度 \(S=3n-6\)

能量按自由度均分原理

分子平均动能 \(\overline{\varepsilon_{kt} }\)

\(\overline{\varepsilon _{kt} }= \frac{3}{2}kT\)

每个平动自由度上都得到相同的平动动能 \(\overline{ {\varepsilon _{kti} } } = \frac{1}{2}kT\)

分子平均动能 \(\overline{\varepsilon_{k} }\) 按自由度划分

能量均分定理: 在一定温度 \(T\) 的平衡态下, 所有物质的分子在每一个自由度上 (平动,转动,整栋) 都有一份相同的平均动能 \(\frac{1}{2}kT\)

\(\overline{\varepsilon_{k} }=\frac{i}{2}kT = \frac{t+r+s}{2}kT\)

分子平均总能量 \(\overline{\varepsilon }\) 按自由度划分

分子总能量 \(\overline{\varepsilon }= \overline{\varepsilon_{k} } + \overline{\varepsilon_p}\)

谐振动在一个周期内的平均振动势能 = 平均振动动能

\(\overline{\varepsilon _{p} }= \overline{\varepsilon_{ks} } = \frac{s}{2}kT\)

\[ \begin{aligned} \overline{\varepsilon } & = \overline{\varepsilon_{k} } + \overline{\varepsilon_{s} } \\ & = \frac{t+r+s}{2}kT + \frac{s}{2}kT \\ & = \frac{t+r+2s}{2}kT \\ \end{aligned} \]

理想气体的内能

分子总自由度 \(i = t+r+2s\) 一个理想气体分子的内能 $ E =kT = kT$ 质量为 \(m\) 的理想气体内能 \(\displaystyle E = \frac{m}{M} \frac{i}{2}RT\)

  • 一定质量的理想气体内能只取决于\(i,T\)
  • 内能是温度的单值函数

麦克斯韦分子按速率分布定理

大量原子分子呈现出的行为: 必然性和规律性

分布函数

  • \(\displaystyle f (x) = \frac{\Delta N_i}{N \Delta x}\)
  • \(\displaystyle f (x) = \frac{\mathrm{d} N_i}{N \mathrm{d}x}\)

麦克斯韦速率分布率

处在温度 \(T\) 的平衡态下的理想气体, 如果在 \(v-v+\mathrm{d}v\) 的速率区间 \(\mathrm{d}v\) 内有 \(\mathrm{d}N\) 个分子

\[ \begin{equation} \frac{\mathrm{d}N_{v}}{N}= 4\pi v^{2} \left( \frac{m}{2 \pi kT} \right)^{\frac{3}{2}} e^{-\frac{m v^{2}}{2kT}}\mathrm{d}v \end{equation} \]

麦克斯韦速率分布函数:

\[ \begin{equation} f (v) = \frac{\mathrm{d}N_v}{N\mathrm{d}v}=4\pi \left( \frac{m}{2 \pi kT} \right)^{\frac{3}{2}} e^{-\frac{m v^{2}}{2kT}}v^{2} \end{equation} \]

处在温度为 \(T\) 的平衡态下的理想气体，在 \(v\) 附近的单位速率区间内的分子数占总分子数的比例.或几率密度

曲线下的一小块面积 \(\displaystyle f (v)\mathrm{d}v = \frac{\mathrm{d}N_v}{N}\) 表示速率在 \([v ,v+\mathrm{d}v]\) 区间内的分子数占总数的比例

速率分布函数的归一化条件 整个曲线下的总面积为1 \[ \int _{0}^{\infty} f (v)\mathrm{d}v = 1 \]

最概然速率

与分布函数 \(f (v)\) 的极大值相对应的速率 \(v_p\) 叫最概然速率

  • 对大量分子而言，在等宽的速率间隔中,气体分子的速率在 \(v_p\) 附近的分子数最多
  • 对单个分子而言，速率在 \(v_p\) 附近的几率最大
  • 速率大和速率小的分子数都比较少，具有中等速率的分子数居多

气体速率分布的实验测定

\[ \frac{L}{v}= \frac{\phi}{\omega} \Rightarrow v = \frac{L \omega }{\phi}\]

选择不同的速率区间 \[ |\Delta v| = \frac{Lw}{\phi^{2}} \Delta \phi = \frac{v}{\phi} \Delta \phi \]

分子的三个特征速率

函数 \(f(v)\) 可以求出分子速率的三个统计平均值 \(\displaystyle v_p,\bar{v}, \sqrt{\overline{v^{2}}}\)

求 \(v_p\) \(\displaystyle \frac{\mathrm{d} f (v)}{dv} = 0\)

\[ v_p = \sqrt{\frac{2kT}{m}} = \sqrt{\frac{2RT}{M}} \approx 1.41 \sqrt{\frac{RT}{M}}\]

\[ \bar{v}=\int \frac{v \mathrm{d}N_v}{N} = \int v f (v) \mathrm{d}v = \sqrt{\frac{8kT}{\pi m}} = \sqrt{\frac{8RT}{\pi M}} > v_p\]

\[ \overline{v^{2}} = \int_{0}^{\infty} v^{2} f(v) \mathrm{d}v = \frac{3kT}{m} \]

\[ \sqrt{\overline{v^{2}}}= \sqrt{\frac{3kT}{m}} = \sqrt{\frac{3RT}{M}} \]

\[ v_p = \sqrt{\frac{2RT}{M}} < \bar{v} = \sqrt{\frac{8RT}{\pi M}} < \sqrt{\overline{v^{2}}} = \sqrt{\frac{3RT}{M}}\]

  • 求速率分布: 最概然速率 \(v_p\)
  • 求平均自由程、平均碰撞频率: 平均速率 \(\bar{v}\)
  • 计算平均平动动能: 方均根速率 \(\sqrt{\overline{v^{2}}}\)

\(g (v)\) 的统计平均值由下式计算

\[ \overline{g (v)}=\frac{\int g (v)\mathrm{d}v}{\int \mathrm{d}N} = \frac{\int g (v) f (v)\mathrm{d}v}{\int f (v)\mathrm{d}v}\]

  • 在 \(v→v+\mathrm{d}v\) 速率区间出现的分子数
    • \(\displaystyle N f (v)\mathrm{d}v =\mathrm{d}N\)
  • 在整个速率区间分子的平均平动动能
    • \(\displaystyle \int_{0}^{\infty}\frac{1}{2}mv^{2} f (v)\mathrm{d}v = \overline{\varepsilon_{t}}\)

玻耳兹曼能量分布定律

\[ w =\frac{N_i}{N} \varpropto e^{-\frac{\varepsilon_i}{kT}}\]

理想气体分子在重力场中按位置的分布

\((x,y,z,v_x,v_y,v_z) \to \\(x +\mathrm{d}x, y+\mathrm{d}y,z+\mathrm{d}z,v_x+\mathrm{d}v_x,v_y+\mathrm{d}v_y,v_z+\mathrm{d}v_z)\)

此区间内分子的总能量为 \[ \begin{aligned} E & = E_k + E_p \\ & = \frac{1}{2}mv^{2} + E_p \\ & = \frac{1}{2}m (v_x^{2} + v_y^{2} + v_z^{2}) \\ \end{aligned} \]

根据玻耳兹曼分布律 \[ \mathrm{d}N =C e^{-\frac{E_k+E_p}{kT}}\mathrm{d}v_x \mathrm{d}v_y \mathrm{d}v_z \mathrm{d}x\mathrm{d}y\mathrm{d}z \]

由此计算体积元 \(\mathrm{d}x\mathrm{d}y\mathrm{d}z\) 内的分子数 \(\mathrm{d}N'\)

\[ \mathrm{d}N' = \left[ \iiint\limits_{-\infty}^{\infty} C e^{-\frac{E_k}{kT}} \mathrm{d}v_x \mathrm{d}v_y\mathrm{d}v_z \right] e^{-\frac{E_p}{kT}}\mathrm{d}x\mathrm{d}y\mathrm{d}z \]

\[ \begin{aligned} \mathrm{d}N' & = C' e^{-\frac{E_p}{kT}}\mathrm{d}x\mathrm{d}y\mathrm{d}z \\ \end{aligned} \] 体积元 \(\mathrm{d}x\mathrm{d}y\mathrm{d}z\) 内的分子数密度: \[ n = \frac{\mathrm{d}N'}{\mathrm{d}x\mathrm{d}y\mathrm{d}z} = C' e^{-\frac{E_p}{kT}} \]

设 \(E_p=0\) 处的分子数密度为 \(n_{0}\),则 \(C' = n_0\)

\[ n = n_0 e^{-\frac{E_p}{kT}} \]

\(m\) 单个分子的质量 \[ n = n_0 e^{-\frac{mgh}{kT}} \]

\[ P = n kT =n_0kT e^{-\frac{mgh}{kT}} \]

\(h=0\) 处压强 \(P_0= n_0kT\)

\[ P = P_0 e^{-\frac{mgh}{kT}} \]

\[ \begin{aligned} \mathrm{d}P & = -mgn_0 e^{\frac{-mgh}{kT}}\mathrm{d}h \\ & = -mgn \mathrm{d}h \\ \end{aligned} \]

\[ P_0S = mgS \int_{0}^{\infty}n\mathrm{d}h = Nmg \]

理想气体在平衡态下分子按状态的概率分布

\[ \varepsilon_{ix} = \frac{1}{2} m v_{ix}^{2} \] 分子速度在区间 \(\displaystyle \left[ v_{ix},v_{ix}+\mathrm{d}v_{x} \right]\)间的概率为:

\[ w_{x} = \frac{N_{ix}}{N} \varpropto e^{-\frac{\varepsilon_{ix}}{kT}} = e^{-\frac{mv_{ix}^{2}}{2kT}} \]

\[ w_x \varpropto e^{-\frac{mv_{ix}^{2}}{2kT}} \mathrm{d}v_x \mathrm{d}x\]

\[ w_y \varpropto e^{-\frac{mv_{iy}^{2}}{2kT}} \mathrm{d}v_y \mathrm{d}y\]

\[ w_z \varpropto e^{-\frac{mv_{iz}^{2}}{2kT}} \mathrm{d}v_z \mathrm{d}z\]

  • \(w\) 表示速率在 \((\vec{r},\vec{v}) \to (\vec{r} + \mathrm{d}\vec{r},\vec{v}+\mathrm{d}\vec{v})\) 内的分子数占总分子数的比例
  • 单个分子 \(w\) 表示其在此状态区间的概率 \[ \begin{aligned} w & = \frac{\mathrm{d}N_{\vec{r},\vec{v}}}{N} \\ & = w_x w_y w_z \\ & = A e^{- \frac{m (v_{x}^{2}+ v_y^{2}+ v_z^{2})}{2kT}} \mathrm{d}v_x \mathrm{d}v_y\mathrm{d}v_z\mathrm{d}x\mathrm{d}y\mathrm{d}z \\ \end{aligned} \]

归一化条件

\[ A \int_{-\infty}^{\infty} \mathrm{d}x e^{- \frac{m (v_{x}^{2}+ v_y^{2}+ v_z^{2})}{2kT}} \mathrm{d}v_x \mathrm{d}v_y\mathrm{d}v_z\mathrm{d}x\mathrm{d}y\mathrm{d}z = 1\]

\[ A = \frac{1}{V} \left( \frac{m}{2\pi kT} \right)^{\frac{3}{2}} \]

推导: \(\displaystyle \int_{-\infty}^{\infty} e^{-x^{2}} \mathrm{d}x= \sqrt{\pi}\)

\[ w = \frac{1}{V} \left( \frac{m}{2\pi kT} \right)^{\frac{3}{2}} e^{- \frac{m (v_{x}^{2}+ v_y^{2}+ v_z^{2})}{2kT}} \mathrm{d}v_x \mathrm{d}v_y\mathrm{d}v_z\mathrm{d}x\mathrm{d}y\mathrm{d}z\]

麦克斯韦速度分布律(与速率分布律区分): \[ \frac{\mathrm{d}N_{\vec{v}}}{N} = \left( \frac{m}{2\pi kT} \right)^{\frac{3}{2}} e^{- \frac{m (v_{x}^{2}+ v_y^{2}+ v_z^{2})}{2kT}} \mathrm{d}v_x \mathrm{d}v_y\mathrm{d}v_z\]

麦克斯韦速度分布函数:

\[ f (\vec{v}) = \left( \frac{m}{2\pi kT} \right)^{\frac{3}{2}} e^{- \frac{m (v_{x}^{2}+ v_y^{2}+ v_z^{2})}{2kT}}\]

x方向分布:

\[ \frac{\mathrm{d}N_{\vec{v_x}}}{N} = \left( \frac{m}{2\pi kT} \right)^{\frac{1}{2}} e^{- \frac{m v_{x}^2}{2kT}} \mathrm{d}v_x\]

\[ \frac{\mathrm{d}N_{\vec{v_x},\vec{v_y}}}{N} = \left( \frac{m}{2\pi kT} \right) e^{- \frac{m (v_{x}^2+ v_y^{2})}{2kT}} \mathrm{d}v_x\mathrm{d}v_y\]

对真实气体的几点简介

分子的平均碰撞次数,平均自由程

分子的平均碰撞频率 \(\overline{\mathbf{Z} }\)

  • 一个分子单位时间内和其他分子平均碰撞的次数
  • \(\overline{\mathbf{Z} } = \sqrt{2} n \pi d^2 \overline{v}\)
  • \(\overline{\mathbf{Z} } = \sqrt{2} n \pi d^2 \sqrt{\frac{8RT}{\pi M} }\)

分子的平均自由程

  • \(\overline{\lambda} = \frac{\overline{v} }{\overline{\mathbf{Z} } }= \frac{1}{\sqrt{2}\pi d^2n}\)
  • \(\overline{\lambda} = \frac{kT}{\sqrt{2}\pi d^2 p}\)
  • 说明
    • 标况下
      • \(\overline{\mathbf{Z} }=7.41 \times 10^9 s^{-1}\)
      • \(\overline{\lambda} = 6.11 \times 10^{-8} m\)

偏离平衡态

非平衡态到平衡态 输运过程(热运动)

常见的气体内的迁移现象

  • 动量的输运
    • 内摩擦或者粘滞现象
      • 粘滞力
        • \(\mathrm{d}f=-\eta \left ( \frac{du}{dy}\right)\mathrm{d}S\)
        • \(\mathrm{d}P_{动量} = \mathrm{d}f\mathrm{d}t = -\eta \left ( \frac{du}{dy}\right)\mathrm{d}S\mathrm{d}t\)
        • \(\eta>0\) 粘滞系数 粘度
        • \(\eta = \frac{1}{3} nm_f\overline{v}\overline{\lambda}\)
  • 能量的输运
    • 热传导
      • \(\mathrm{d}Q = -\kappa\frac{dT}{dy}\mathrm{d}S\mathrm{d}t\)
      • \(\kappa>0\) 热导率
      • \(\kappa = \frac{1}{3}nm_f\overline{v}\overline{\lambda}\frac{C_{V,m} }{M} \quad M:气体摩尔质量\)
  • 质量的输运
    • 扩散
      • \(\mathrm{d}m = -D \frac{d\rho}{dy}\mathrm{d}S\mathrm{d}t\)
        • \(D\)为扩散系数
        • \(D=\frac{1}{3}\overline{v}\overline{\lambda}\)

理想气体 \(P=nKT\) 在几十个大气压内,近似符合

非理想气体状态方程:范德瓦尔斯方程

理想气体微观模型:忽略了分子的大小,引力

分子体积的修正

  • 分子的有效直径\(10^{-10}m\)
  • 1mol气体的固有体积:
    • \(N_A \times \frac{4}{3}\pi r^3 \approx2.5\times10^{-6}(m^3)\)
  • \(V=\frac{4}{3}\pi d^3 = 8\times \frac{4}{3}\pi r^3\)
  • 分子最紧密状态必须占有的空间为 \(\frac{1}{2}V = 4 \times \frac{4}{3}\pi r^3即固有体积的四倍\)
  • b:1mol最紧密气体的体积
  • \(P(V_m -b) = RT\)
  • \(b \approx 10^{-5} m^{-3}\)

分子引力的修正

  • 引力的有效作用半径 \(r_e= 10^-9 m\)
  • \(P= \frac{RT}{V_m -b} - \vert \Delta P\vert\)
  • \(\vert\Delta P\vert= \frac{a}{V_m^2}\)
    • 靠近薄壁的分子会受到指向气体的拉力
  • \((P+\frac{a}{V_m^2})(V_m-b)=RT\)
  • \((P+\frac{m^2}{M^2}\frac{a}{V^2})(V-\frac{m}{M}b) = \frac{m}{M}RT\)
