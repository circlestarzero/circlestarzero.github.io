热力学基础

热力学第一定律

功、热量、内能

热力学系统与外界传递能量的两种方式

  • 做功
  • 传热
  • 功(A)
    • 能量传递和转化的量度；是过程量。
    • 系统对外做功:A>0
    • 外界对系统做功:A<0
  • 热量(Q)
    • 是传热过程中所传递能量的多少的量
  • 内能(E)

功、热量、内能的关系

功与内能的关系

\((E_2 - E_1) = A_外\)

  • 外界对系统作功,无传热
  • 内能的改变量可以用绝热过程中外界对系统所作的功来量度

热量与内能的关系

外界与系统之间不作功,仅传递热量 \(Q =(E_2 -E_1)\)

  • 在外界不对系统作功时，内能的改变量也可以用外界对系统所传递的热量来度量
  • 作功和传热效果一样

热力学第一定律

\(\Delta E= A _{外} + Q \quad E:系统的内能\) * 仅与过程的初末态有关,与过程无关

\(A _{外} = -A\) * 外界对系统的功A外与系统对外界的功A等值反号

\(Q = \Delta E +A\)

  • 热力学第一定律

  • \(Q>0\) 系统从外界吸热,\(Q<0\) 系统向外界放热

  • \(A>0\) 系统对外做正功,\(A<0\) 系统对外做负功

  • 适用范围,任何热力学系统的任何过程 \(\mathrm{d}Q = \mathrm{d}E +\mathrm{d}A\)

  • 热力学第一定律微分形式

意义阐释

  • 外界对系统传递的热量 \(Q\),一部分用于系统内能增加,另一部分用于系统对外做功

  • 热力学第一定理是能量转化与守恒定律在热现象中的具体体现

    • 广义下: \(Q = \Delta E + W\)
      • \(\Delta E\) 包含机,电,化学等
      • \(W\)包含所有功

  • 第一类永动机不可能实现

    • 以机械的手段使体系持续地向外输出能量

功与热量的表达式

准静态过程的具体表达式

功的表达式

  • 体积功 \[ \mathrm{d} A = PS \mathrm{d} l = P \mathrm{d} v \]

系统的体积由 \(V_1\) 变到 \(V_2\),系统对外做功: \[ A = \int_{}^{} \mathrm{d} A = \int_{V_1}^{V_2}P \mathrm{d} v \]

功不仅与初末状态有关,还与过程有关,是 过程量 \(\mathrm{d} A\) 不是状态参量的全微分

理想气体做功 $(P_1,V_1) 等温地准静态变化到 (P_2,V_2) $

\[ A = vRT \ln \frac{V_2}{V_1} \]

热容量及热量的表达式

物体温度升高1k所需要吸收的热量,称为该物体的热容量 \(C\), 单位为 \(J/k\) \[ C = \frac{\mathrm{d} Q}{\mathrm{d} T} \]

热容量与系统的质量 (摩尔数) 以及经历的过程有关

  • 定压摩尔热容
    • \[ C _{P,m} = \frac{1}{v} \left(\frac{\mathrm{d} Q}{\mathrm{d} T}\right) _{P} \]
    • \[ C _{V,m} = \frac{1}{v} \left(\frac{\mathrm{d} Q}{\mathrm{d} T}\right) _{V} \]

系统吸收热量 \[ \left \{ \begin{aligned} \mathrm{d} Q =& C \mathrm{d} T\\ Q = &\int_{T_1}^{T_2} C \mathrm{d} T \end{aligned} \right. \]
