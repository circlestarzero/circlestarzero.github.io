shortcuts

发表于 2022-05-01 阅读次数： Valine：
本文字数： 2.1k 阅读时长 ≈ 2 分钟

ctrl + M 行内公式

ctrl + / 将当前行注释或反注释

ctrl + shift + alt + E 计算表达式,用 = 连接

ctrl + shift + alt + R 计算表达式,并替换

ctrl + shift + alt + D 定义表达式,无输出

ctrl+ alt + v 粘贴并保存图片

shift+option+ ↑ 向上复制当前行

option+ ↑ 向上移动当前行

option+ ↓ 向下移动当前行

ctrl + D当前有选中文本时, 将下一个与其相同的文本加入选中 (重要)

Shift + Ctrl + Alt + C 转化为alinged模式
